# minishell

Minishell is a project at 42 US

Objectives:
  Reimplement a Unix shell from scratch
  Understand how to fork and wait for processes
  Manage and execute successfully from the bin
  recreate system builtins:
    \nenv
    cd
    echo
    unsetenv
    setenv
    
Supports
   cd -
   cd
   cd ~
   echo $ANYVAR
